## About

I'm a full stack web developer with 5+ years of experience building business, healthcare, and SaaS websites — from marketing pages to secured admin panels with authentication and role-based access. I work end-to-end: architecture, frontend, backend, and deployment.

Recent work includes a multi-phase admin panel (Next.js + MySQL) with TOTP two-factor authentication and SMS/WhatsApp OTP delivery, and production sites for healthcare diagnostics and robotics clients.

**Currently:** freelancing full-time, building [sohelmalek.com](https://sohelmalek.com) and client projects across Next.js, WordPress, and custom Tailwind builds.
